# Public Folder Structure

Place your assets in the `public` folder:

## Profile Photo
- Save as: `public/profile.jpg` or `public/profile.png`
- Recommended size: 800x1000px (3:4 aspect ratio)
- Update Hero.tsx to use the image

## Project Videos
Create a videos folder and add your demo videos:
- `public/videos/voice-agent-demo.mp4`
- `public/videos/email-automation-demo.mp4`
- `public/videos/udaash-demo.mp4`

Video specifications:
- Format: MP4
- Codec: H.264
- Max resolution: 1920x1080
- Duration: 10-30 seconds
- Keep file size under 5MB for optimal performance

## Icons and Other Assets
- Favicon: `public/favicon.ico`
- Open Graph image: `public/og-image.jpg`
